# EkippX Documentation

EkippX is a C++20 macro-expansion and text-transformation library with a CLI,
REPL integration, plugin API, batteries-included macro package, and generated
API reference from Doxygen comments.

## API Reference

The generated reference covers `EkippX.hpp`, `EkippX-Batteries.hpp`,
`EkippX-PluginAPI.hpp`, compatibility plugin headers, plugin examples, and CLI
translation units. Doxygen uses this file as the main page and includes the
manual chapters as related pages.
