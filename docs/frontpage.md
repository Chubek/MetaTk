# MetaTk Documentation

MetaTk is the umbrella C++20 workspace for DSLtk, Parzek, and EkippX. The
workspace documentation combines generated API references with project manuals
for parser combinators, parser generation, macro expansion, CLI integration, and
plugin-oriented extension points.

## API Reference

The generated reference indexes public headers, implementation translation
units, examples, and Markdown manuals from the enabled subprojects. Doxygen uses
this page as the main page and emits HTML, LaTeX, XML, and installable PDF-ready
LaTeX artifacts.
